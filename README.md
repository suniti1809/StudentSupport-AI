## 🚀 Overview

StudentSupport AI is a **Full-Stack Generative AI web application** designed to enhance students' learning experience through intelligent AI assistance.

The application integrates the **Groq API** with the **Llama 3.1 Large Language Model (LLM)** to generate accurate, context-aware, and personalized responses for academic queries. Instead of relying on predefined answers, the AI understands user prompts and produces natural-language responses in real time.

This project demonstrates practical implementation of **Generative AI**, **Prompt Engineering**, **REST APIs**, and **Full-Stack Web Development**.

## 🧠 How Generative AI Works

1. The user enters an academic question.
2. React sends the request to the Flask backend.
3. Flask forwards the prompt to the Groq API.
4. Llama 3.1 processes the request and generates an intelligent response.
5. The AI-generated answer is displayed instantly in the chat interface.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/suniti1809/studentsupport-ai.git
cd studentsupport-ai
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
```

### Frontend Setup

```bash
cd ../frontend
npm install
npm run build
```

### Run the Application

```bash
cd ../backend
python main.py
```

Open your browser and visit:

```
http://localhost:8000
```
